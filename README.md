⏱ Timer App (SwiftUI)
A simple yet visually engaging countdown timer built with SwiftUI. This app allows users to set a custom timer, track progress through an animated circular ring, and switch between day and night themes for a better user experience.

## Features include:
1.Customizable countdown timer
2.Animated progress indicator
3.Start, pause, and reset functionality
4.Light/Dark (day/night) theme toggle
5.Sound notification when the timer ends

This project was created to practice SwiftUI fundamentals, state management, and building interactive UI components.
